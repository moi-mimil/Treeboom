## 📉 Mission Objective
In **Treeboom**, your goal is simple: **Total Environmental Collapse**.  
The world starts clean and vibrant, and it is your job to drive the CO2 levels up by systematically destroying every tree in sight.  

---

## 🎮 Gameplay Mechanics
You control **Miku**, armed with endless **Damazon** delivery boxes. Use them to strike down the nature around you.  

| Feature | Details |
|---------|---------|
| 🎯 Goal | Aim and throw boxes at trees. Every tree hit increases `co2_level`. |
| 🔄 Evolution | As CO2 rises (Stages 12, 22, 32), the world transforms from lush forest → grey industrial wasteland. |
| ⚔️ Obstacle | <u>Anti-Milo</u> is the guardian of the grove. He will hunt you. Touch him → game over. |
| ⚠️ Hazard | Avoid water! Stay on land to continue your mission. |


---

## 🕹️ Controls
| Action | Key |
|--------|----|
| 🏃 Movement | `Z/Q/S/D` (AZERTY) / `W/A/S/D` (QWERTY) |
| 🥊 Attack | Left Click to launch a Damazon box |
| 📦 Ammo | Maximum of 3 boxes in flight |

---

## 🛠️ Technical Stack
Built with **Python** and **Pygame**, featuring:

- 🎯 **Vector-based AI:** Anti-Milo tracks your position using normalized direction vectors.
- ⚡ **Dynamic Difficulty:** The higher your score, the faster Anti-Milo becomes.
- 💾 **Data Persistence:** High scores and mute settings saved in `controls.txt`.

---

## 📦 Download Instructions

### Requirements
- **Python 3.9+**
- **Pygame 2.0+**

Modules included with Python (no install required): `sys`, `math`, `subprocess`, `time`, `random`.

### How to Play
Run the following commands in a terminal:

```bash
git clone https://github.com/moi-mimil/Treeboom.git
cd Treeboom
pip install pygame
python treeboom.py
````
